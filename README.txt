# Challenge Oracle ONE - Crea tu Propio Encriptador de Texto

¡Bienvenido al Challenge Oracle ONE!

## Descripción
En este desafío, tendrás la oportunidad de poner a prueba tus habilidades en HTML, CSS y JavaScript creando tu propio encriptador de texto. El objetivo es diseñar y desarrollar una aplicación web que permita a los usuarios encriptar y desencriptar mensajes de texto utilizando un algoritmo específico.

## Instrucciones
### Algoritmo de Encriptación
El encriptador sigue las siguientes reglas de conversión:
- La letra "e" se convierte en "enter"
- La letra "i" se convierte en "imes"
- La letra "a" se convierte en "ai"
- La letra "o" se convierte en "ober"
- La letra "u" se convierte en "ufat"

### Requisitos
- El encriptador debe funcionar exclusivamente con letras minúsculas del alfabeto inglés.
- No deben utilizarse letras con acentos ni caracteres especiales en el texto de entrada.
- El encriptador debe permitir la conversión bidireccional, es decir, convertir una palabra al formato encriptado y también revertir una palabra encriptada a su forma original.

#### Ejemplos
- "gato" => "gaitober"
- "gaitober" => "gato"

### Funcionalidad de la Página
- La página web debe incluir campos de entrada de texto para el mensaje a encriptar o desencriptar.
- Los usuarios deben tener la opción de elegir entre encriptar o desencriptar el mensaje.
- El resultado encriptado o desencriptado debe ser mostrado en pantalla de manera clara y legible.

### Extras (Opcionales)
- Agregar un botón que permita copiar el texto encriptado/desencriptado al portapapeles, similar a la función "copiar" del menú de las aplicaciones.

## Cómo Empezar
1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web para acceder a la aplicación.
3. Sigue las instrucciones en la interfaz para encriptar y desencriptar mensajes.
